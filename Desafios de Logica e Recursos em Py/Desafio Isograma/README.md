# Desafio: Identificar as palavras que são isogramas.

## O que é um isograma?
Um isograma é uma palavra onde nenhuma letra se repete, como: "amor", "tempo", "bravo", "colher" e etc.
Exemplos não-isogramas: "ana", "ovo", "teto", "mamão", "papel".
